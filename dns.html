Tutorial de instalacion servidor DNS

Marzo 25, 2023 posted por Juan E. Castro

El sistema de nombres de dominio (DNS) es el directorio telefónico de Internet. Las personas acceden a la información en línea a través de nombres de dominio como nytimes.com o espn.com. Los navegadores web interactúan mediante direcciones de Protocolo de Internet (IP). El DNS traduce los nombres de dominio a direcciones IP para que los navegadores puedan cargar los recursos de Internet. A continuacion se relizara un tutorial de como se realiza la intalacion de un servidor de DNS en Windows Server 2012.

Requisitos:

  • Direccion Ip estatica
  • Maquina virtual Windows Server 2012

Paso 1: Cofiguracion de Ip estatica.

Se configura la Ip estatica de la maquina virtual para poder instalar el servidor de DNS.

Paso 2: Instalacion del servidor DNS.

Se realiza la instalacion del servidor de DNS a traves del Administrador del servidor en la opcion Agregar Roles y Caracteristicas.

Paso 3: Configuracion del servidor DNS.

Se realiza la configuracion del servidor de DNS en la seccion herramientas del Administrador del Servidor en la opcion DNS. Nos situaremos primero en la carpeta de Zona de busqueda directa.

Paso 4: Creacion de Zona de busqueda directa.

Se crea una nueva zona de busqueda directa dando click derecho en la carpeta, en este caso la zona se creara con el nombre de juanes.com. Las demas opciones se dejan por defecto siguiendo la intalacion sugerida.

Paso 5: Creacion de Zona de busqueda inversa.

Asi mismo como se creo la zona de busqueda directa, se crea una zona de busqueda invera dando click derecho sobre la carpeta con el mismo nombre en la cual se nos pedira una direccion Ip. Para el caso teniendo en cuenta la configuracion de Ip estatica que se realizo al comienzo esta sera 192.168.4.

Paso 6: Creacion de host en la zona de busqueda directa.

Entrando en la carpeta zonas de busqueda directa entramos a la zona que creamos, damos click derecho en la zona blanca y creamos un nuevo host. El host llevada como nombre www y la direccion Ip de la maquina que configuramos de manera estatica. Para el caso la direccion Ip es 192.168.4.10.

Pruebas de funcionamiento del servidor DNS.

Para realizar la prueba de que el servidor esta bien configurado, se ejecuta el comando ping en la consola del sistema (cmd) a la direccion www.juanes.com si muestra el ping significa que el servidor quedo configurado correctamente.
